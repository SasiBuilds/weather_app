# 🌦 WeatherSphere

WeatherSphere is a modern weather dashboard built using HTML, CSS, and JavaScript. It provides real-time weather information, location-based forecasts, air quality monitoring, and an intuitive glassmorphism-inspired user interface.

The application allows users to search for weather conditions in any city, view detailed weather metrics, access hourly and 5-day forecasts, and monitor air quality data through a responsive and user-friendly dashboard.

---

## ✨ Features

### 🌍 Real-Time Weather

* Search weather by city name
* Display current weather conditions
* Live weather data

### 📍 Location-Based Weather

* Detect user's current location
* Fetch weather information automatically using Geolocation API

### 🌡 Weather Details

* Temperature
* Weather Condition
* Humidity
* Wind Speed
* Atmospheric Pressure
* Visibility

### ⏰ Forecasting

* Hourly Weather Forecast
* 5-Day Weather Forecast
* Dynamic Weather Icons

### 🌫 Air Quality Monitoring

* Air Quality Index (AQI)
* Real-time pollution data

### 🎨 User Experience

* Glassmorphism User Interface
* Dark Mode Support
* Responsive Design
* Loading Animation
* Dynamic Weather Backgrounds

### 💾 Local Storage

* Search History Storage
* Theme Preference Persistence
